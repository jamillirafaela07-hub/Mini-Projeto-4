Sistema de Organização de Estudos

Este projeto é um sistema para estudantes do ensino médio organizarem seus estudos. Ele permite registrar conteúdos estudados, acompanhar revisões, adicionar resumos e marcar o progresso de cada matéria.  
O objetivo é ajudar estudantes a manterem um estudo mais eficiente e organizado, especialmente quem se prepara para provas como o ENEM.

Funcionalidades
- Adicionar conteúdos estudados
- Marcar status do conteúdo (não iniciado / em andamento / ok)
- Excluir conteúdos
- Adicionar resumos

Tecnologias
- Frontend: (React, HTML/CSS, JS)
- Backend/API: json-server
- Operações REST: GET, POST, PATCH, DELETE

Como Rodar o Projeto
- Clone o repositório:  
git clone <https://github.com/jamillirafaela07-hub/Mini-Projeto-4.git>
- Instale as dependências:
npm install
- Inicie o json-server:
npx json-server --watch db.json --port 3000
- Inicie o frontend:
npm start
